# Analog TV Testing Method:

Signal source can be commercial TV broadcasting signal 

or **HackTV** with commands as below:

ffmpeg -i input.mp4 -vf scale=832:576 output.mp4

hacktv -f 434000000 -m i -g 47 output.mp4
